HybridRetriever — parallel async fetch

async def retrieve(query, project_id, top_k=150): bm25_task = asyncio.create_task(self.bm25.search(query, project_id, top_k)) dense_task = asyncio.create_task(self.vector.search(query, project_id, top_k)) bm25_hits, dense_hits = await asyncio.gather(bm25_task, dense_task) return bm25_hits, dense_hits
2.3
RRFMerger — Reciprocal Rank Fusion
Merge the two ranked lists using the RRF formula. k=60 is fixed per spec. Output is a single unified list of (chunk_id, rrf_score, payload) tuples.
def merge(bm25_hits, dense_hits, k=60): scores = defaultdict(float) for rank, hit in enumerate(bm25_hits): scores[hit.id] += 1 / (k + rank + 1) for rank, hit in enumerate(dense_hits): scores[hit.id] += 1 / (k + rank + 1) # sort descending, attach payloads return sorted(scores.items(), key=lambda x: x[1], reverse=True)

3.1
Gate 1 — similarity score threshold
Read the raw cosine similarity from the top dense hit. If below configurable threshold (default 0.5), short-circuit the entire pipeline immediately — no reranking, no LLM call.
GATE1_THRESHOLD = float(os.getenv("GATE1_THRESHOLD", "0.5")) def check_threshold(dense_hits: list[DenseHit]) -> RefusalEvent | None: if not dense_hits or dense_hits[0].score < GATE1_THRESHOLD: return RefusalEvent(level=1, reason="level_1_threshold", message="Not found in the document.") return None
3.2
Gate 2 — calibrated reranker score
After NeuralReranker runs, check the top reranker score. Calibrate threshold by running 20–30 known-bad queries against the Adani PDF and finding the score that separates them from good queries. Store threshold in env var.

4.1
NeuralReranker class — CrossEncoder
Use cross-encoder/ms-marco-MiniLM-L-6-v2 (fast) or Zerank-1 (accuracy). Run batch inference over top-150 (query, chunk_text) pairs. Sort by score descending.
from sentence_transformers import CrossEncoder model = CrossEncoder("cross-encoder/ms-marco-MiniLM-L-6-v2") def rerank(query: str, chunks: list[RetrievedChunk], top_k: int = 20): pairs = [(query, c.raw_text) for c in chunks] scores = model.predict(pairs) # numpy array ranked = sorted(zip(chunks, scores), key=lambda x: x[1], reverse=True) for chunk, score in ranked[:top_k]: chunk.reranker_score = float(score) return [c for c, _ in ranked[:top_k]]
